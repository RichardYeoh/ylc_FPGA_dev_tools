# 文本转 UTF-8 工具使用说明书

## 1. 工具简介
本工具用于将工程目录下的文本文件自动转换为 UTF-8 编码。仅当文件内容包含中文且当前编码不是 UTF-8 时才会进行覆盖写入。

## 2. 运行环境
- Windows
- Python 3（推荐 3.8 及以上）
- 无需额外依赖

## 3. 启动方式
在 `text_code_to_utf8` 目录中执行：

```powershell
python text_to_utf8_tool.py
```

或双击运行：

`一键启动.bat`

## 4. 使用步骤
1. 点击“选择目录”，选择需要处理的工程目录。
2. 如需处理无后缀文件，可勾选“包含无后缀文件（若为文本）”。
3. 点击“开始转换”，确认弹窗后开始处理。
4. 日志窗口实时显示每个文件处理结果与统计信息。
5. 处理完成后弹出“完成”提示框。

## 5. 日志说明
日志中会输出：
- `[已转换]`：已从非 UTF-8 转为 UTF-8
- `[已是UTF-8]`：已经是 UTF-8
- `[无中文]`：未检测到中文，跳过
- `[疑似二进制]`：二进制文件跳过
- `[解码失败]`：候选编码均无法正确解码

## 6. 注意事项
- 工具会直接覆盖文件，请在重要项目上先自行备份。
- 仅处理文本文件（源代码、脚本、文档、配置等），不会处理明显的二进制文件。
